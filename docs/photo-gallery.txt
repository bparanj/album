Photo Gallery using Lightbox 2 in Rails 5

Download [lightbox 2](http://lokeshdhakar.com/projects/lightbox2/ 'lightbox2'). 
Extract the zip file. 
Copy the lightbox.min.css to app/vendor/assets/stylesheets directory.
Copy the lightbox.min.js to app/vendor/assets/javascripts directory.

Add
//= require lightbox

to application.js

Add

*= require lightbox

in application.css

Copy the close.png, loading.gif, next.png and prev.png to app/assets/images folder.


app/assets/javascripts/application.js

//= require jquery
//= require jquery_ujs
//= require turbolinks	
//= require lightbox
//= require_tree .

app/assets/stylesheets/application.scss

 *= require_tree .

app/views/layouts/application.html.erb

<!DOCTYPE html>
<html>
  <head>
    <title>Album</title>
    <%= csrf_meta_tags %>

    <%= stylesheet_link_tag 'application', media: 'all', 'data-turbolinks-track': 'reload' %>
  </head>

  <body>
    <div id="container">
      <% flash.each do |name, msg| %>
        <%= content_tag :div, msg, id: "flash_#{name}" %>
      <% end %>
      <%= yield %>
    </div>
	
    <%= javascript_include_tag 'application', 'data-turbolinks-track': 'reload' %>
	
  </body>
</html>

Javascript must be just before the body tag ends, otherwise the photo gallery will not work.

Browse to http://localhost:3000/photos to view the lightbox photo gallery in action. We can now remove the hard coded images to the uploaded images like this:

<div>
  <% @photos.each do |photo| %>	  	  
    <%= link_to(image_tag(attachment_url(photo, :image, :fit, 100, 100, format: 'jpg')), attachment_url(photo, :image, :fit, 500, 500, format: 'jpg'), data: { lightbox: "example-set", title: photo.name }) %>		
  <% end %>
</div>	  

This will display the uploaded images in a gallery.
